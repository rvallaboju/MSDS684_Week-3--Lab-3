# MSDS684 Week 2 Lab 3

## Overview
This repository contains materials for the Week 2 Lab 3 of the MSDS684 course. The Jupyter notebook included demonstrates various data analysis techniques using Python.

## Setup Instructions
To set up the project, follow these steps:

1. **Clone the Repository**:  
   Use the following command to clone the repository:
   ```bash
   git clone https://github.com/rvallaboju/MSDS684_Week-2--Lab-3.git
   ```

2. **Navigate to the Project Directory**:  
   ```bash
   cd MSDS684_Week-2--Lab-3
   ```

3. **Install Requirements**:  
   Ensure you have Python installed. Then, create a virtual environment and install the required packages:
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows use `venv\Scripts\activate`
   pip install -r requirements.txt
   ```

## Using the Jupyter Notebook
The Jupyter notebook in this repository provides a hands-on experience with data analysis. It includes examples and exercises that help you learn and apply techniques in data science.


## Important Note
Make sure to run the notebook in the virtual environment where the requirements are installed to avoid any package-related issues.
